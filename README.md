For convert number base 10 to HEX and BIN.

First step (one time)

make re && make fclean

After : 

Usage : ./convert [number_base_10]

Exemple :

Serveur_root_AP $ ./convert 255
[255] >> BIN : 11111111
[255] >> HEX : FF

other Exemple : 

Serveur_root_AP $ ./convert 100
[100] >> BIN : 1100100
[100] >> HEX : 64
